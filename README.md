# ES6 Phaser + Mocha tests

ES6 Boilerplate with Phaser, Mocha &amp; Babel

*To set up*

```
npm install
npm run build-init // racks up phaser in dist dirs
```

*To run build*

```
npm run build
```

*To run tests*

`npm run test`

This will:

 - run mocha tests in spec
 - build es6 -> common using babel
